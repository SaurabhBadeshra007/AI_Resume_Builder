# AI Resume Generator – Full Stack Project

This project is an AI-powered Resume Generator built using HTML, CSS, and JavaScript. The application allows users to enter their personal details, education, skills, experience, and projects, and automatically generates a professional ATS-friendly resume.

## 🚀 Features

- AI-based Resume Generation
- Clean and Professional Resume Format
- ATS-Friendly Resume Structure
- Resume Preview Section
- Download Resume Option
- Copy Resume HTML Feature
- Responsive User Interface
- Easy-to-use Form Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- AI API Integration
- DOM Manipulation
